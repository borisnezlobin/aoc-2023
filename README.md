## dev setup

- create venv

```
python3 -m venv venv
```

- activate venv

windows

```
venv\Scripts\activate.bat
```

unix/macos

```
source venv/bin/activate
```

To deactivate the virtual environment, type `deactivate` in the respective terminal window.

## creating a new day's folder

```
bash makeday.sh day01
```
